# tetrizz
A remake on tetris using bun testing and exporting functions

# Deploy
To run, use:
```js
    bunx vite
```
or:
```js
    npm install vite
    node vite
```
or check out a deployed version on deno here: https://tetrizz-benca.deno.dev/

# Test

To test, run:
```js
    bun test --watch // hot reloads on save
```
